# RN-2021

 Для запуска приложения необходимо:
 - Получить ключ от АПИ https://rapidapi.com/skyscanner/api/skyscanner-flight-search/endpoints.
 - Создать в папке src файл api_key.json
 - Вставить туда этот ключ. 
 Пример: 
 {
  "RapidapiKey": "API-KEY"
 }
